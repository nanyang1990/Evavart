CD++ MODELER

Introduction

CD++ Modeler is a tool for designing and executing DEVS Models in an integrated enviroment

In this application you can design atomic models made of nodes, in-ports, out-ports, internal and external links beetween them

You can design coupled models made of other predifined atomic models, coupled models, in-ports, out-ports and links beetween them

You can execute the defined models using the integrated cd++ server

You can animate the simulation in three diferent ways
