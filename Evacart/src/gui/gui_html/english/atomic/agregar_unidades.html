Adding Units

To add a new State to the Atomic Model double-click on the canvas. States are automatically created with a name. In this case newUnit. Adding new States is explained in Adding Atomic State Units chapter

 

To add a Port, right-click on the canvas. Ports are automatically created with a name. Adding new Port Units is explained in Adding Port Units chapter

 

To add a Link click on the origin of it and drag until the destiny of the new Link. The Link will be created only if both, origin and destiny are valid. In this case a Link was created from newUnit0 to newUnit3.
