Agregar Unidades

Para agregar un Puerto, seleccione el Puerto en el Panel lateral superior y hacer doble-click sobre el Panel de Modelado

 

 

Los puertos se crean un nombre de forma automática. En este caso, Port0 es un Puerto de Entrada y Port1 es un Puerto de salida.

Para agregar un nuevo Modelo Atomico hacer click sobre el boton "Add new Atomic Model Unit" de la botonera.

Los Modelos se crean en la esquina superior izquierda del Panel de Modelado. Se crean con un nombre automático. En este caso newAtomicModel2

 

Para agregar un nuevo Modelo Acoplado hacer click sobre el boton "Add new Coupled Model Unit" de la botonera.

Los Modelos se crean en la esquina superior izquierda del Panel de Modelado. Se crean con un nombre automático. En este caso newCoupledModel3

 

 

Para agregar un Link hacer click en el origen del Link y Drag hasta el destino del Link. El Link se creará solamente si el origen y el destino son validos. En este caso se creó un Link desde Port0 a newAtomicModel2

 

 

Para crear un Self Link que vaya de un Modelo a si mismo, hacer click en el modelo de origen, drag hasta una cierta distancia, drag hasta el origen nuevemente y soltar el boton del mouse. El CD++ Modeler le preguntará si quiere crear un Self Link.
