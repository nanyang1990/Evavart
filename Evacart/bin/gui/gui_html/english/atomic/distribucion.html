The Atomic Models editor of CD++Modeler application is composed by a central panel, 2 lateral panels and an upper button bar.

 

 

On the central panel appear the units of the Atomic Model. It can be added Input Ports, Output Ports, States and Links between them.

On the upper lateral panel they can be seen the objects that form the Atomic Model that is being defined. It is divided in States, Links, Ports And Variables.

On the lower lateral panel a description of the element selected on the upper lateral panel.

The button bar has the following buttons:

  • New

    Creates a new Atomic Model

  • Open

    Opens a previously saved Atomic Model.

  • Save

    It saves this Atomic Model

  • Help

    It brings up this help

  • Internal/Extenal

    Defines the type of link to be created

  • Add new Atomic Model Unit

    Not used in this view

  • Add new Coupled Model Unit

    Not used in this view
